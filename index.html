Báo Cáo: Fibonacci

Tìm hiểu cách tính số Fibonacci bằng Python với hàm đệ quy.


def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
print(fibonacci(10))

Hệ Thống Gợi Ý

Giới thiệu hệ thống gợi ý đơn giản dùng cosine similarity.


import pandas as pd
from sklearn.metrics.pairwise import cosine_similarity

def recommend(user_id, data):
    similarity = cosine_similarity(data)
    return similarity[user_id]
